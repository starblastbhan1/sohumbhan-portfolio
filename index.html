Robotics Projects

Robotic Hand Version 1.0

Python Raspberry PI 3D Printing CAD

The focus of this project was to actuate fingers on a robotic hand to follow motion movements. The fingers were designed in CAD software, 3D-printed, and connected to motors. The motors were controlled by a glove with sensors, allowing the robotic hand to mimic the movements of the hand inside the glove. The connectivity between the sensors and the motors was done in Python on a Raspberry PI.

Robotic Hand Version 2.0

Electronic Circuit FSM IMU 3D Printing

The focus of this project is to make a robotic hand entirely in hardware. To control the behavior of the hand, I used electronic circuits. I created a variable PWM generation circuit that is used to control a servo motor, which adjusts the finger position. I used a series of state machines to create a number of presets for the hand, that could be quickly switched to or altered.

Computer Vision Pick & Place with Kinematics

OpenCV 6 DOF ROS Kinematics

Created a forward and inverse kinematic model for a 6 DOF robotic arm. Used OpenCV to design a color sorting algorithm that picks up blocks and places them in groups by color using the kinematics models, with feedback and correction for errors like dropped blocks.

Drone Controller for path following

Python PID OpenCV ROS

This project involved creating a controller for a drone to follow a predefined path, correcting for deviations from it, and avoiding obstacles during travel. The PID controller used a state-based design, focusing on low overshoot and minimal steady-state error. The vision controller combined a neural network and color masking to detect obstacles within a frame, using the color masking for high-framerate pose estimation and the neural network for slower but more accurate analysis. These two controllers are combined and assigned various intensities based on what is needed at the time.
